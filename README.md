# Tensorflow View,一个基于tensorflow的可视化平台
### 说明
Tensorflow View（一个开箱即用的机器学习工具），降低机器学习门槛，使得开发者只需要关注数据源，切换配置项就可训练模型提供给自己使用。
* PS:此项目基于flask和Vue开发本人负责需求分析，项目设计，日常开发
* 前后端端代码开源持续更新。
